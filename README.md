# 🚀 What is Prompt Engineering?

**Prompt Engineering** is the process of designing effective prompts to get accurate, useful, and high-quality responses from Large Language Models (LLMs) like ChatGPT, Gemini, Claude, and Copilot.

> “Small changes in prompts can unlock massive improvements in AI output.”

# 🧩 Core Prompt Engineering Techniques

## 1️⃣ Be Specific

### ❌ Weak Prompt
```text
Write something about AI.
```

### ✅ Strong Prompt
```text
You are an AI educator.
Write a beginner-friendly 200-word explanation of Artificial Intelligence for BCA students.
Use simple language and give one real-world example.
```

---

## 2️⃣ Role Prompting

Give the AI a role before asking the task.

```text
You are a senior software engineer.
Explain REST APIs in simple terms.
```

---

## 3️⃣ Few-Shot Prompting

Provide examples so the model learns the pattern.

```text
Input: Login page error
Output: [Auth] Login Failure

Input: PDF upload crashes app
Output: [Upload] PDF Upload Crash
```

---

## 4️⃣ Chain of Thought (CoT)

Ask the AI to think step by step.

```text
Solve this math problem step by step before giving the final answer.
```

---

## 5️⃣ Structured Output

Request JSON, Markdown, XML, or Tables.

### Example
```json
{
  "title": "AI Basics",
  "difficulty": "Beginner",
  "topics": ["ML", "NLP", "Robotics"]
}
```

---

## 6️⃣ Constraints & Rules

Tell AI what NOT to do.

```text
Write a professional email.
Do not use emojis.
Keep it under 100 words.
```

---

## 7️⃣ Iterative Refinement

Improve results through multiple prompts.

```text
Make it shorter.
More professional.
Add one example.
```

---

# 🔥 Advanced Prompting

## 🧠 Interview Style Prompting

Let the AI ask questions before generating output.

```text
Before writing the blog post,
ask me questions one by one to collect all required information.
```

---

## 🔗 Prompt Chaining

Break complex tasks into smaller prompts.

### Example Workflow
```text
Step 1 → Generate Outline
Step 2 → Expand Sections
Step 3 → Create SEO Meta Description
```

# 📋 Example Professional Prompt

```text
You are a senior technical writer.

Create a beginner-friendly tutorial on Git and GitHub for BCA students.

Requirements:
- Use simple English
- Add examples
- Include commands
- Keep tone friendly
- Use markdown formatting
- Add conclusion section
```

# ⚠️ Common Mistakes

| Mistake | Fix |
|---------|-----|
| Too Vague | Add details |
| Too Many Tasks | Split prompts |
| No Examples | Use few-shot prompting |
| No Format | Request structured output |
